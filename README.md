# Airbnb Clone Project Overview
This is a clone of the popular accommodation booking platform

# Goal
The goal of this project is to build a web application that allow users to find and book listed properties

# Tech stack
Frontend: HTML, CSS, JavaScript (React or similar framework)
Version Control: Git and GitHub
Design Tools: Figma for UI/UX design

  # Project Roles and Responsibilities.
  1. Project manager
    Roles and Responsibilties
    1. Planning project timelines,
    2. Coordinating communication among all team members
    3. Managing risks,
    4. Ensuring the team stays on schedule
    5. Reporting project progress to stakeholders

  2. Product Owner (PO)
     Responsibilities
     1. Defining product vision and goals
     2. Prioritizing features based on business value
     3. Providing clarifications about requirements

  3. Scrum Master
   Responsibilities
   1. Facilitating daily standups, sprint reviews, and retrospectives
   2. Removing blockers for the team
   3. Coaching team in Agile best practices
   4. Shielding the team from external interruptions
   5. Ensuring good communication within the team

   4. Frontend Developers
    Responsibilites 
    1. Building responsive UI using HTML, CSS, JS/TS
    2. Implementing designs from UI/UX designers
    3. Integrating with backend APIs
    4. Ensuring cross-browser and cross-device compatibility
    5. Optimizing performance and user experience

    5. Backend Developers
     Responsibilities
     1. Designing and building APIs
     2. Implementing business logic
     3. Managing databases and data models
     4. Ensuring system security and scalability
     5. Integration with third-party services
     6. Writing documentation for backend services
